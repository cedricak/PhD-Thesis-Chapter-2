# PhD-Thesis-Chapter-2

This repository contains the Python code accompanying Chapter 2 of my PhD thesis. The notebook implements the statistical modelling framework and numerical experiments developed in the chapter.

To ensure reproducibility while respecting data confidentiality constraints, synthetic datasets are generated directly within the code to mimic the structure of the original data. These datasets are then used to reproduce the estimation procedures and computational experiments discussed in the chapter.

The repository illustrates the implementation of advanced statistical models and numerical methods using Python, including data simulation, model estimation, and analysis workflows.
